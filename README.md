# Treasure Vaultr Portfolio

Static portfolio website for Treasure, a video editor and motion designer.

## Deploy Folder

The deployable site uses:

- `index.html`
- `styles.css`
- `script.js`
- `fonts/`
- `Logos/`
- `portfolio videos web/`
- `portfolio thumbnails/`

Do not deploy the original `portfolio videos/` folder. It contains large master files and is intentionally ignored.
