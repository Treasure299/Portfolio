# Video Swap Guide

To replace a portfolio video later, export the new video as an `.mp4`, then give it the exact same filename as the video you want to replace and put it inside the `portfolio videos` folder.

Before publishing, create a compressed full-length web version and put it inside `portfolio videos web` with the same filename. The website uses the compressed files from `portfolio videos web`, while `portfolio videos` keeps the original high-quality masters safe.

The site also uses still thumbnail images from `portfolio thumbnails`. When you replace a video, make a new `.jpg` thumbnail with the same base filename. For example, `Podcast intro.mp4` should have `Podcast intro.jpg`.

Current video filenames used by the site:

- `special motion graphics video.mp4` - the large background banner in the "click" section. If this file is missing, the site uses `Motion design video (recreated w my own touch).mp4` as a fallback.
- `Motion design concept (Lemfi).mp4` - hero and Motion Design Concepts.
- `Motion design concept (Netflix).mp4` - hero, Motion Design Concepts, selected work, and footer motion.
- `Motion design video (recreated w my own touch).mp4` - hero, Motion Design Concepts, and fallback banner.
- `Podcast intro.mp4` - Podcast Intro project.
- `Podcast Short-form content.mp4` - Podcast Social Cut project.
- `Talking head video (edited as a beginner).mp4` - Film & Story Editing.
- `Wedding video 1 (edited as a begginner).mp4` - Film & Story Editing and selected work.

For free hosting, keep each compressed full-length web video around 22-24 MB or less when possible. This is the current balance used here: full-length playback, visibly better quality, and still small enough for common free static hosting limits. Avoid uploading the original multi-hundred-MB or GB files directly to the website.
